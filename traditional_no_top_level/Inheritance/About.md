# About
Abstract classes vs Interface

---

## When to use which?

When to use abstract class:
- many repeating fields
- teacher is Person, student is Person
- dog is Animal, cat is Animal
- ```IS```

When to use interface:
- repeating actions that can be done by several classes
- salary can Pay, Invoice can Pay
- ```CAN```
